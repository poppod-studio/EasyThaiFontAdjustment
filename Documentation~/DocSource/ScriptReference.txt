HOW IT WRITES TO THE FONT
-------------------------
Adjustments are written as Glyph Pair Adjustment Records into
TMP_FontAsset.fontFeatureTable.glyphPairAdjustmentRecords.

The record element type differs across versions. The tool aliases it
via versionDefines so it compiles on both:
   - TMP 3.0.6 (Unity 2021/2022): TMPro.TMP_GlyphPairAdjustmentRecord
   - TMP 3.2+ / UGUI 2.0 / Unity 6: UnityEngine.TextCore.LowLevel.GlyphPairAdjustmentRecord
